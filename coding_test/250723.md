## **문제 1 : [flag에 따라 다른 값 변환하기](https://school.programmers.co.kr/learn/courses/30/lessons/181933)**

### 문제내용
> 두 정수 a, b와 boolean 변수 flag가 매개변수로 주어질 때, flag가 true면 a + b를 false면 a - b를 return 하는 solution 함수를 작성해 주세요.

### 문제풀이
```python
def solution(a, b, flag):
    return a + b if flag else a - b

```

## **문제 2 : [배열 만들기 5](https://school.programmers.co.kr/learn/courses/30/lessons/181912)**

### 문제내용
> 문자열 배열 intStrs와 정수 k, s, l가 주어집니다. intStrs의 원소는 숫자로 이루어져 있습니다. 배열 intStrs의 각 원소마다 s번 인덱스에서 시작하는 길이 l짜리 부분 문자열을 잘라내 정수로 변환합니다. 이때 변환한 정수값이 k보다 큰 값들을 담은 배열을 return 하는 solution 함수를 완성해 주세요.

### 문제풀이
```python
def solution(intStrs, k, s, l):
    answer = []
    for i in intStrs:
        num = int(i[s:s+l])
        if k < num:
            answer.append(num)
    return answer
```
> 문자열 숫자에 int변환시 생기는 변화와 문자열 슬라이싱 인덱스에 대한 이해가 필요

## **문제 3 : [로그인 성공?](https://school.programmers.co.kr/learn/courses/30/lessons/120883)**

### 문제내용
> 머쓱이는 프로그래머스에 로그인하려고 합니다. 머쓱이가 입력한 아이디와 패스워드가 담긴 배열 id_pw와 회원들의 정보가 담긴 2차원 배열 db가 주어질 때, 다음과 같이 로그인 성공, 실패에 따른 메시지를 return하도록 solution 함수를 완성해주세요.

> * 아이디와 비밀번호가 모두 일치하는 회원정보가 있으면 "login"을 return합니다.
> * 로그인이 실패했을 때 아이디가 일치하는 회원이 없다면 “fail”를, 아이디는 일치하지만 비밀번호가 일치하는 회원이 없다면 “wrong pw”를 return 합니다.
### 문제풀이 
```python
def solution(id_pw, db):
    id_to_pw = {id:pw for id, pw in db}

    check_id = id_to_pw.get(id_pw[0])
    if not check_id :
        return 'fail'
    else :
        if check_id == id_pw[1] :
            return 'login'
        else :
            return 'wrong pw'
```
>1. 분기점을 어떻게 잡아야할지 계획 세우기.
>2. get을 통한 dictionary value값 호출하면 일반적인 호출과의 차이점 알기(get을 통해서 호출했을 때 해당하는 value가 없다면 None을 반환하지만 []을 통해 호출 했을 때 해당하는 key 값이 없으면 error가 발생한다.

## **문제 4 : [홀짝에 따라 다른 값 반환하기](https://school.programmers.co.kr/learn/courses/30/lessons/181935)**

### 문제내용
> 양의 정수 n이 매개변수로 주어질 때, n이 홀수라면 n 이하의 홀수인 모든 양의 정수의 합을 return 하고 n이 짝수라면 n 이하의 짝수인 모든 양의 정수의 제곱의 합을 return 하는 solution 함수를 작성해 주세요.

### 문제풀이
```python
def solution(n):
    if n % 2 == 1:
        return (n // 2 + 1)**2
    else :
        return sum([i**2 for i in range(2, n+ 1, 2)])
```
>1. 문제에서 의도한 바는 listcomprehension을 통한 range와 함께 분기점별 연산값을 내놓기를 원한 것으로 보여진다.
>2. 모든 홀수의 합은 특정 정수에 제곱인 것이 잘 알려져 있어 이를 이용해 홀수인 경우에 리스트컴프리헨션을 이용하지 않아 연산량이 많이 줄일 수 있다.

## **문제 5 : [등차수열의 특정한 향만 더하기](https://school.programmers.co.kr/learn/courses/30/lessons/181931)**

### 문제내용
> 두 정수 a, d와 길이가 n인 boolean 배열 included가 주어집니다. 첫째항이 a, 공차가 d인 등차수열에서 included[i]가 i + 1항을 의미할 때, 이 등차수열의 1항부터 n항까지 included가 true인 항들만 더한 값을 return 하는 solution 함수를 작성해 주세요.

### 문제풀이
```python
def solution(a, d, included):
    answer = 0
    # a + d * (i - 1) 등차수열
    for ind, i in enumerate(included):
        if i:
            result = a + d*(ind + 1 - 1)
            answer += result

    return answer
```

>1. 등차수열의 식을 이해 필요 $a_{n} = a_{1} + d*(n - 1)$
>2. 문제에서 index를 이용해서 답을하기를 요구하기 때문에 enumerate를 이용해 index 호출

## **문제 6 : [치킨 쿠폰](https://school.programmers.co.kr/learn/courses/30/lessons/120884)**

### 문제내용
> 프로그래머스 치킨은 치킨을 시켜먹으면 한 마리당 쿠폰을 한 장 발급합니다. 쿠폰을 열 장 모으면 치킨을 한 마리 서비스로 받을 수 있고, 서비스 치킨에도 쿠폰이 발급됩니다. 시켜먹은 치킨의 수 chicken이 매개변수로 주어질 때 받을 수 있는 최대 서비스 치킨의 수를 return하도록 solution 함수를 완성해주세요.

### 문제풀이
```python
def solution(chicken):
    answer = 0
    while chicken // 10 > 0:
        answer += chicken // 10
        chicken = chicken % 10  + chicken//10
    return answer
```

>1. 나머지와 몫 계산에 끝판왕
>2. chicken = coupon이므로 coupon을 10으로 나눈 몫이 서비스 치킨
>3. 서비스 치킨을 시키면 coupon 차감되므로 나머지로 chicken 갱신
>4. 서비스로 시킨 치킨도 coupon이 발행되므로 몫과 나머지를 더한 값으로 chicken을 갱신
>5. 더 이상 서비스치킨을 시킬 수 없을 때 종료 chicken > 10 이라는 조건으로 대체해도됨

## **문제 7 : [두 수의 연산값 비교하기](https://school.programmers.co.kr/learn/courses/30/lessons/181938)**

### 문제내용
> 연산 ⊕는 두 정수에 대한 연산으로 두 정수를 붙여서 쓴 값을 반환합니다. 예를 들면 다음과 같습니다.
> * 12 ⊕ 3 = 123
> * 3 ⊕ 12 = 312
> 양의 정수 a와 b가 주어졌을 때, a ⊕ b와 2 * a * b 중 더 큰 값을 return하는 solution 함수를 완성해 주세요. 
> 단, a ⊕ b와 2 * a * b가 같으면 a ⊕ b를 return 합니다.

### 문제풀이
```python
def solution(a, b):
    calcul1 = int(str(a) + str(b))
    calcul2 = 2*a*b
    return calcul1 if calcul1 >= calcul2 else calcul2
```

## **문제 9 : [저주의 숫자 3](https://school.programmers.co.kr/learn/courses/30/lessons/120871)**

### 문제내용
> 3x 마을 사람들은 3을 저주의 숫자라고 생각하기 때문에 3의 배수와 숫자 3을 사용하지 않습니다. 3x 마을 사람들의 숫자는 다음과 같습니다.
> |10진법 |3x 마을에서 쓰는 숫자|10진법|3x 마을에서 쓰는 숫자|
> |:---:|:---------:|:------:|:-----:|
> |1|	1|	6|	8|
> |2|	2|	7|	10|
> |3|	4|	8|	11|
> |4|	5|	9|	14|
> |5|	7|	10|	16|

> 정수 n이 매개변수로 주어질 때, n을 3x 마을에서 사용하는 숫자로 바꿔 return하도록 solution 함수를 완성해주세요.

### 문제풀이
```python
def solution(n):
    answer = 0
    for i in range(n):
        answer += 1
        while answer % 3 == 0 or '3' in str(answer):
            answer += 1
    return answer
```
>1. 처음에는 현재 while문에 사용되어져 있는 조건문을 for문을 통해 loop를 돌때 조건문에 해당하는 값을 더해 원래 n에 더해주려고 했다.
>2. 1.에 풀이는 잘못되어진게 3x 마을에서 사용되는 숫자로 변환하면서 n이 커지면서 그사이에 발생하는 조건들이 반영이 되지 않는다는 것
>3. 이를 해결하기 위해 while을 통해 조건에 해당하면 숫자를 건너 뛰게 구조를 만듦
