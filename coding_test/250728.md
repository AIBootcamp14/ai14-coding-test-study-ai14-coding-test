## **문제 1 : [외계행성의 나이](https://school.programmers.co.kr/learn/courses/30/lessons/120834)**

### 문제내용
> 우주여행을 하던 머쓱이는 엔진 고장으로 PROGRAMMERS-962 행성에 불시착하게 됐습니다. 입국심사에서 나이를 말해야 하는데, PROGRAMMERS-962 행성에서는 나이를 알파벳으로 말하고 있습니다. a는 0, b는 1, c는 2, ..., j는 9입니다. 예를 들어 23살은 cd, 51살은 fb로 표현합니다. 나이 `age`가 매개변수로 주어질 때 PROGRAMMER-962식 나이를 return하도록 solution 함수를 완성해주세요.



### 문제풀이
```python
def solution(age):
    age = str(age); answer = ''
    alien_age = {i:'abcdefghijklmnopqrstuvwxyz'[i] for i in range(10)}
    for i in age:
        answer += alien_age[int(i)]
    return answer
```
> 숫자를 넣으면 외계행성의 숫자를 반환해주는 dictionary를 만들고 loop를 통해 변환

## **문제 2 : [문자열 여러 번 뒤집기](https://school.programmers.co.kr/learn/courses/30/lessons/181913)**

### 문제내용
> 문자열 `my_string`과 이차원 정수 배열 `queries`가 매개변수로 주어집니다.<br> `queries`의 원소는 [s, e] 형태로, `my_string`의 인덱스 s부터 인덱스 e까지를 뒤집으라는 의미입니다. `my_string`에 `queries`의 명령을 순서대로 처리한 후의 문자열을 return 하는 solution 함수를 작성해 주세요.

### 문제풀이
```python
def solution(my_string, queries):
    for s, e in queries:
        temp = ''
        temp += my_string[:s]
        temp += my_string[s:e+1][::-1]
        temp += my_string[e+1:]
        my_string = temp
    return my_string
```
>1. indexing을 이용해 한번에 처리하려고 했지만 일반화하기에 너무 어려운 부분이 있어 나누어서 실행 해주었음
>2. 다른 사람들의 풀이를 보아하니 문자열을 list화 하여 마지막에 join하는 방식이 가장 간결하고 좋은 답안인듯
