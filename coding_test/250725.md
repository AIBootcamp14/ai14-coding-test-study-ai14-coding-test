## **문제 1 : [n 번째 원소부터](https://school.programmers.co.kr/learn/courses/30/lessons/181892)**

### 문제내용
> 정수 리스트 `num_list`와 정수 `n`이 주어질 때, `n` 번째 원소부터 마지막 원소까지의 모든 원소를 담은 리스트를 return하도록 solution 함수를 완성해주세요.

### 문제풀이
```python
def solution(num_list, n):
    return num_list[n - 1:]
```
> 기초 인덱스 슬라이싱

## **문제 2 : [A로 B 만들기](https://school.programmers.co.kr/learn/courses/30/lessons/120886)**

### 문제내용
> 문자열 `before`와 `after`가 매개변수로 주어질 때, `before`의 순서를 바꾸어 `after`를 만들 수 있으면 1을, 만들 수 없으면 0을 return 하도록 solution 함수를 완성해보세요.

### 문제풀이
```python
def solution(before, after):
    after = ''.join(sorted(after))
    before = ''.join(sorted(before))

    return 1 if after == before else 0
```
>1. 원소를 하나하나 체크하면서 제거해나가는 방식을 제일 먼저 생각했지만, 하나하나 찾는 과정의 시간이 너무 오래 걸릴 가능성이 있다. 최악의 경우 n 부터 1까지 전부 곱하는 시행횟수를 겪어야 해결될 수도 있다
>2. 정렬을 통해 after, before를 정렬 기준을 통해 나열하고 같은지 확인하는 과정을 통해 간단히 해결 가능

## **문제 3 : [특이한 정렬](https://school.programmers.co.kr/learn/courses/30/lessons/120880)**

### 문제내용
> 정수 `n`을 기준으로 `n`과 가까운 수부터 정렬하려고 합니다. 이때 `n`으로부터의 거리가 같다면 더 큰 수를 앞에 오도록 배치합니다. 정수가 담긴 배열 `numlist`와 정수 `n`이 주어질 때 `numlist`의 원소를 n으로부터 가까운 순서대로 정렬한 배열을 return하도록 solution 함수를 완성해주세요.

### 문제풀이 
```python
def solution(numlist, n):
    return sorted(numlist, key = lambda x : (abs(x - n), -x))
```
>1. 정렬 기준을 바꿔주는 파이썬 내장함수 sorted에 key를 이용해서 문제에서 요구하는 정렬을 해준다.
>2. `abs(x-n)`은 요구하는 `n`에서의 상대적인 거리를 의미한다.
>3. 첫번째 조건인 `abs(x-n)`이 같다면 같은 조건의 숫자 중 큰 숫자가 먼저 호출되어야하므로 두번째 정렬기준을 선정해주는데, 그냥 x로 선정하면 작은 것부터 오름차순으로 정렬되므로 -x를 기준으로 정렬기준을 정의한다.
