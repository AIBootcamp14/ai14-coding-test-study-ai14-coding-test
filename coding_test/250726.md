## **문제 2 : [배열의 길이를 2의 거듭제곱으로 만들기](https://school.programmers.co.kr/learn/courses/30/lessons/181857)**

### 문제내용
> 정수 배열 `arr`이 매개변수로 주어집니다. `arr`의 길이가 2의 정수 거듭제곱이 되도록 `arr` 뒤에 정수 0을 추가하려고 합니다. `arr`에 최소한의 개수로 0을 추가한 배열을 return 하는 solution 함수를 작성해 주세요.

### 문제풀이
```python
def solution(arr):
    length = len(arr); n = 0;
    while length > 2**n:
        n += 1
    added_zero = (2 ** n) - length
    arr.extend([0] * (added_zero))
    return arr
```

### 다른 문제 풀이
```python
def solution(arr):
    length = len(arr)
    import math
    num = math.log2(length)
    if int(num) != num :
        num = 1 + int(num)
    arr.extend((int(2 ** num) - length) * [0])
    return arr
```
>1. `arr` 보다 큰 2의 승수를 알려고 while문을 통해 승수를 알고 extend를 통해 문제에서 요구하는 0을 채워넣는 방식
>2. 다른 문제 풀이에서는 `math` 내장 라이브러리를 통해 log2 를 통해 arr에 길이가 2의 몇승수인지 확인
>3. 2의 몇승수 인지 확인하는 과정이 while문을 통한 확인보다 훨씬 경제적

## **문제 3 : [배열 조각하기](https://school.programmers.co.kr/learn/courses/30/lessons/181893)**

### 문제내용
> 정수 배열 `arr`와 `query`가 주어집니다. <br> `query`를 순회하면서 다음 작업을 반복합니다. 

> * 짝수 인덱스에서는 `arr`에서 `query[i]`번 인덱스를 제외하고 배열의 `query[i]`번 인덱스 뒷부분을 잘라서 버립니다.
> * 홀수 인덱스에서는 `arr`에서 `query[i]`번 인덱스는 제외하고 배열의 `query[i]`번 인덱스 앞부분을 잘라서 버립니다.

> 위 작업을 마친 후 남은 `arr`의 부분 배열을 return 하는 solution 함수를 완성해 주세요.

### 문제풀이 
```python
def solution(arr, query):
    for ind, i in enumerate(query):
        if ind % 2 == 0:
            arr = arr[:i + 1]
        else :
            arr = arr[i:]
    return arr
```
>1. 실행하는 `query` list에loop를 했을 때 원소와 index를 모두 반환하는 파이썬 내장함수인 `enumerate`를 이용하여 문제를 푼다
>2. indexing slicing을 이용해 arr을 갱신해준다.
